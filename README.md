# 🛡️ Gemini Privacy & Navigation

![Demo](https://drive.google.com/uc?export=view&id=16eDFY3igOma9daPzQH4jfAPmCzZ16moD "Demo")

這是一個專為 **Google Gemini** 網頁版設計的全方位增強擴充功能（Chrome / Edge）。

它結合了 **隱私防護 (Privacy)** 與 **使用效率 (Utility)**。除了能自動模糊側邊欄與敏感圖片外，還新增了右側的「對話導航目錄」，並將功能開關完美整合進 Gemini 的原設定選單中。

## ✨ 主要功能

### 🔒 隱私與防護
* **⏳ 長按解鎖 (Hold to View)**：
    * 防止滑鼠誤觸！採用 **5 秒延遲解鎖** 機制。
    * 滑鼠懸停時，底部出現 **藍色讀取條**，讀取完畢後自動解除模糊。
    * **嚴格點擊防護**：在遮罩未消失前，點擊操作會被攔截，防止誤點擊內容。
* **📂 側邊欄智慧折疊**：
    * 側邊欄 (`My Stuff`) 預設強制折疊為 **40px** 高度，保持介面極致整潔。
    * 長按解鎖後，會自動平滑展開完整內容。
* **🖼️ 全面圖片防護**：
    * 支援主畫面與列表中的 `library-item-card` 圖片卡模糊。
    * **智慧衝突排除**：若圖片卡位於側邊欄內，會自動取消雙重遮罩，避免視覺衝突。

### 🚀 導航與介面優化
* **💬 智慧對話導航 (Smart TOC)**：
    * 在畫面右側自動生成 **「對話導航目錄」**。
    * 自動抓取你的提問，點擊即可平滑捲動至該對話位置。
    * **自動隱藏**：剛進入頁面或無對話時自動隱藏，不干擾視線。
* **⚙️ 原生整合開關**：
    * **無縫整合**：直接在 Gemini 左下角的 **「設定」** 選單中新增 **「NSFW 模糊遮罩」** 選項。
    * **記憶功能**：自動記住你的開關狀態，下次開啟瀏覽器時保持設定。
    * **原生外觀**：完全模仿 Gemini 的 UI 風格（Material Design），包含圖示與 Checkbox。
* **🎨 雙主題自動適配**：
    * 完美支援 **深色模式 (Dark)** 與 **淺色模式 (Light)**。
    * 導航列與遮罩會根據 Gemini 的主題自動切換顏色（黑玻璃 / 白玻璃風格）。

## 📂 檔案結構

專案資料夾應包含以下檔案：

* `manifest.json` - 擴充功能設定檔 (V3, 權限設定)。
* `style.css` - 視覺樣式表 (CSS 變數、模糊、折疊動畫)。
* `script.js` - 核心邏輯 (導航生成、開關注入、狀態管理)。
* `icon.png` - 擴充功能圖示 (建議 128x128)。
* `README.md` - 專案說明文件。

## 🚀 安裝教學

此為個人開發工具，請使用瀏覽器的「開發人員模式」安裝：

1.  **準備檔案**：
    * 將所有檔案放入資料夾（建議命名為 `gemini-privacy-navigation`）。
2.  **開啟擴充功能管理頁面**：
    * **Chrome**: 輸入 `chrome://extensions`。
    * **Edge**: 輸入 `edge://extensions`。
3.  **啟用開發模式**：
    * 開啟右上角的 **「開發人員模式」 (Developer mode)**。
4.  **載入擴充功能**：
    * 點擊 **「載入未封裝項目」 (Load unpacked)**。
    * 選擇你的專案資料夾。
5.  **完成！**
    * 回到 [gemini.google.com](https://gemini.google.com) 並 **重新整理頁面 (F5)** 即可生效。

## 🛠️ 自定義調整

你可以透過編輯程式碼來微調體驗：

* **調整等待時間**：
    * 開啟 `style.css`，搜尋 `5s`，將其改為你想要的秒數（例如 `2s`）。
* **調整模糊強度**：
    * 開啟 `style.css`，修改 `backdrop-filter: blur(20px);` 的數值。
* **預設開關狀態**：
    * 開啟 `script.js`，修改 `let isNsfwEnabled` 的預設邏輯。
